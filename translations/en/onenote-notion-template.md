# ⚡ Energy Management (Mandatory Before Studying)

> ⚠️ Energy > Time

- [ ] Did I sleep well?
- [ ] Am I at my cognitive peak?
- [ ] Phone out of the room?
- [ ] Environment organized?
- [ ] Water nearby?

If you are not at your peak, postpone for 30 minutes.

---

# ⏱️ Execution Strategy (Mandatory)

## 🔥 Pomodoro (4×25 + 5)

- [ ] 25 min deep focus
- [ ] 5 min break (NO phone)
- [ ] 25 min deep focus
- [ ] 5 min break
- [ ] 25 min practical execution
- [ ] 5 min break
- [ ] 25 min active review + summary

> 💡 Break = walk, water, stretch
> 🚫 Phone use compromises cognitive consolidation

---

# 🔁 Interleaving (Mandatory)

- [ ] Define a second topic to alternate during the week
- [ ] Ensure no topic is studied for more than 2 days in a row
- [ ] Alternate theory + practice

# 2️⃣ Research (Smart Curation)

> Rule: Less is more.

## 📘 Selection

- [ ] Maximum 3 main resources
- [ ] 1 official documentation source as priority
- [ ] 1 main lab
- [ ] 1 complementary material

## 📅 Planning

- [ ] Define the order of consumption
- [ ] Estimate time per resource
- [ ] List what will NOT be studied now

## 🤖 AI

- [ ] Copilot to find official docs
- [ ] Perplexity to gather feedback on the most popular courses
- [ ] NotebookLM to consolidate PDFs
- [ ] Copilot Chat notebook for documents, SharePoint, and internal links with your company's general/confidential classification
- [ ] ChatGPT to compare approaches

## 📌 Deliverable

- [ ] Final curated list with defined order

---

# 3️⃣ Priming (Cognitive Preparation — 30 to 40 min)

> Goal: prepare the brain before going deeper.

## 🧩 Tasks

- [ ] Create an executive summary (10-15 lines)
- [ ] Create a glossary (min. 10 terms)
- [ ] Create 10 guiding questions
- [ ] List 3 real scenarios
- [ ] Create initial flashcards

## 🤖 AI (Mandatory)

- [ ] NotebookLM or Copilot → structured summary
- [ ] NotebookLM → flashcards
- [ ] NotebookLM → audio (podcast), quiz, and infographic

## 📌 Deliverable

- [ ] Saved summary
- [ ] Flashcards created
- [ ] Audio listened to at least once

---

## 🧪 Validation

- [ ] Create a quiz (min. 10 questions)
- [ ] Create an answer key
- [ ] Map anti-patterns
- [ ] Resolve doubts in official sources
- [ ] Explain the topic without consulting the material

## 🤖 AI

- [ ] Copilot to generate quiz
- [ ] Copilot as a mentor
- [ ] NotebookLM for reinforcement

## 📌 Deliverable

- [ ] Quiz + answer key
- [ ] List of common mistakes
- [ ] Updated mind map
